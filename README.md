# Digital Focus Database

A relational database project designed for a digital focus and productivity tracking system.

## Features
- User management
- Focus session tracking
- App blocking system
- Goal and reward tracking
- Daily productivity summaries
- Notification system

## Technologies Used
- SQL Server
- SSMS
- SQL

## Database Concepts
- Primary Keys
- Foreign Keys
- Constraints
- Relational Database Design
- Default Values

## Purpose
This project was created to practice relational database design and SQL development by building a productivity-focused system structure.
